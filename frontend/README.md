# Frontend

Frontend app for Quasar EduSaarthi.

## Run

```powershell
cd frontend
npm install
npm run dev
```

## Build

```powershell
npm run build
```

## API Base URL

Set `VITE_API_URL` if you want a non-default backend URL.

Default: `http://localhost:8000`
